# Patches

Este diretório preserva os patches produzidos e aplicados durante a refatoração
incremental do projeto. Os arquivos históricos podem conter revisões, correções
e tentativas intermediárias; por isso, não devem ser reaplicados em sequência
sobre a versão `2.0.0`.

O patch final da consolidação é:

- `0032-consolidacao-v2.0.0.patch`: versão, documentação, auditoria, migração e
  preparação da release final.

A presença dos patches no repositório tem finalidade de rastreabilidade e
estudo do processo de desenvolvimento. O histórico Git continua sendo a fonte
principal para reconstrução das alterações.
