# Código legado

Este diretório preserva os arquivos C# da implementação anterior à refatoração.
O estado histórico completo também permanece disponível pela tag `v1.0.0`.

Os arquivos deste diretório não pertencem aos projetos da solução .NET 9 e não
serão compilados. Eles são mantidos exclusivamente para consulta, comparação e
rastreabilidade didática.

A nova implementação será desenvolvida em `src/` e testada em `tests/`.
