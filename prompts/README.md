# Prompts

Este diretório preserva os prompts utilizados para conduzir a refatoração
incremental do projeto. Eles documentam objetivos, restrições, branches e
prefixos de patches empregados entre o inventário do legado e a consolidação
da versão `2.0.0`.

O Prompt 32, presente nas versões revisadas do roteiro incremental, encerra o
escopo funcional da refatoração e orienta a release `v2.0.0`.
